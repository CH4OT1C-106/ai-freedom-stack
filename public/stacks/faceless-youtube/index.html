Getting Started (This Weekend)

  1. Pick your niche. Top faceless niches: personal finance, AI/tech explainers, history, psychology, “top 10” lists, motivation. Pick one you can talk about for 100 videos.
  2. Set up Claude. Create a system prompt: “You are a YouTube scriptwriter for a [niche] channel. Scripts should be 8-12 minutes of spoken content with visual cues in brackets.”
  3. Generate your first script. Ask Claude for a script on a trending topic in your niche. Include hook, body, CTA.
  4. Record voiceover in ElevenLabs. Paste the script, pick a voice, download the MP3.
  5. Generate visuals. Use the visual cues from your script to prompt Stable Diffusion or Midjourney. Aim for 15-25 images per video.
  6. Assemble in CapCut or DaVinci Resolve. Drop in narration, layer visuals on the timeline, add auto-captions, background music, transitions.
  7. Upload and optimize. Use Claude to write your title, description, and tags.

Expected output: 2-3 videos/week. Each video takes 3-5 hours start to finish once you have a rhythm.
